<h3>Getting started</h3>

<h4>Requirements</h4>
    - qemu-system-x86_64 

<h4>Build and Run</h4>
this will build the project and run it with the suggested qemu flags

```sh
git clone https://github.com/leulshawel/OS101
make run
```
you can also try this to just build the image file

```sh
make os.img
```
